# CHIP8 Emulator

CHIP8 Emulator written in C

## Dependecies

* [raylib](https://github.com/raysan5/raylib)

## How to run

**GUI mode**

```bash
make run
```

will run the emulator and show the IBM logo on the screen.

**Debug mode**

```bash
make debug
```

will run the emulator with custom debug.ch8 file and print to stdout
